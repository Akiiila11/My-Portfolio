A simple and interactive **Portfolio Website** built with `React.js`, showcasing my projects and skills.  
This portfolio is responsive, visually clean, and highlights my front-end development work.

## Tech Stack

* React.js
* JavaScript
* TailwindCSS
* HTML
* CSS

## Features

* Responsive design for all devices
* Clean and modern UI
* Projects showcase with live demos
* Easy navigation
